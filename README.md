## 👩🏼‍💻 Installation 👩🏼‍💻

1. **Clone the repository:**

    ```bash
    git clone https://github.com/rhyzJ/mission3.git​
    ```

2. **Install dependencies:**

    ```bash
    npm i
    ```

3. **Run the program from Frontend folder:**

    ```bash
    npm run dev
    ```
    
4. **Start server in Backend Folder:**

    ```bash
    npm run dev
    ```
